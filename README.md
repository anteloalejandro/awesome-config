# AwesomeWM configuration

## Dependencies
- `awesome`
- `rofi`

## Recommended packages
These are packages that are not required but are used by default. You may have to change the default settings ( located at `main/user-variables`) to make this configuration work correctly if you do not want to install them.

- Compositor: `picom`
- Web Browser: `firefox`
- Text Editor: `neovim`
- File Manager: `pcmanfm`
- Terminal Emulator: `alacritty`
- Video and Music Player: `mpv`
- Image Viewer: `feh`
- Theme Selector: `lxappearance`
- Screen Settings: `arandr`

## Installation instrucctions
- Install dependencies and, optionally, the recommended packages
- Run `git clone https://gitlab.com/anteloalejandro/awesome-config ~/.config/awesome`
- Link or copy the folders in `external_configs` to `~/.config`
- Install and set the [Pop!\_OS GTK theme](https://github.com/pop-os/gtk-theme).
